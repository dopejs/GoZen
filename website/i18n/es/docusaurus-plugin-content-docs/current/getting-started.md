# Primeros Pasos

## Instalación

Instalar con el script de una línea:

```bash
curl -fsSL https://raw.githubusercontent.com/dopejs/gozen/main/install.sh | sh
```

Desinstalar:

```bash
curl -fsSL https://raw.githubusercontent.com/dopejs/gozen/main/uninstall.sh | sh
```

## Primera Ejecución

Añade tu primer proveedor:

```bash
zen config add provider
```

Iniciar con el perfil predeterminado:

```bash
zen
```

Iniciar con un perfil específico:

```bash
zen -p work
```

Iniciar con un CLI específico:

```bash
zen --cli codex
```
