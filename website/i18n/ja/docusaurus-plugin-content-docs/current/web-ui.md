# Web インターフェース

ブラウザからすべての設定を視覚的に管理できます。必要に応じてデーモンは自動起動します。

## 使い方

```bash
# ブラウザで開く（必要ならデーモンを自動起動）
zen web
```

## 機能

- プロバイダーとプロファイルの管理
- プロジェクトバインディングの管理
- グローバル設定（デフォルトクライアント、デフォルトプロファイル、ポート）
- 設定同期の管理
- 自動更新付きのリクエストログビューア
- モデル入力欄のオートコンプリート

## セキュリティ

デーモンを初回起動したときにアクセス用パスワードが自動生成されます。ローカル以外（127.0.0.1 / ::1 外部）からのリクエストにはログインが必要です。

- セッションベース認証（有効期限を設定可能）
- 指数バックオフ付きブルートフォース対策
- 機密トークン転送時の RSA 暗号化（API キーはブラウザ内で暗号化）
- ローカルアクセス（127.0.0.1）は認証をバイパス

### パスワード管理

```bash
# Web UI パスワードをリセット
zen config reset-password

# Web UI から変更
zen web  # Settings → Change Password
```
