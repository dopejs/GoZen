# 웹 인터페이스

브라우저에서 모든 설정을 시각적으로 관리할 수 있습니다. 필요할 때 데몬이 자동으로 시작됩니다.

## 사용법

```bash
# 브라우저에서 열기 (필요 시 데몬 자동 시작)
zen web
```

## 기능

- 제공자와 프로필 관리
- 프로젝트 바인딩 관리
- 전역 설정 (기본 클라이언트, 기본 프로필, 포트)
- 설정 동기화 관리
- 자동 새로고침이 있는 요청 로그 뷰어
- 모델 입력 필드 자동완성

## 보안

데몬을 처음 시작할 때 접근 비밀번호가 자동 생성됩니다. 로컬이 아닌 요청(127.0.0.1 / ::1 외부)은 로그인해야 합니다.

- 만료 시간을 설정할 수 있는 세션 기반 인증
- 지수 백오프를 적용한 무차별 대입 방지
- 민감한 토큰 전송용 RSA 암호화 (API 키는 브라우저에서 암호화)
- 로컬 접근(127.0.0.1)은 인증 우회

### 비밀번호 관리

```bash
# Web UI 비밀번호 재설정
zen config reset-password

# Web UI에서 변경
zen web  # Settings → Change Password
```
